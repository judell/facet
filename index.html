Click HTML, CSV, or JSON to search for matching Hypothesis annotations and display them in one of those formats. Click to save the current HTML, CSV, or JSON view. Fill in one or more facets to filter results. The facets are username, group, url (or wildcard_uri), tag, and any. If you need more than 50 results, set max to a larger number. If you specify no facets other than the default group All, you'll get recent Hypothesis annotations and replies in any group you're authorized to see. Click to launch the Hypothesis thread viewer/editor. Use exactTagSearch to match tags exactly. Use addQuoteContext to show the prefix and suffix cpatured with each highlight. Results are grouped by URL, ordered by most-recently-updated annotation for each URL, and displayed as threads. Use subject user tokens to enable in-place editing of annotations by users who have shared their tokens with you, and controlled tags to constrain the choices for the first tag belonging to each in-situ-editable annotation.
